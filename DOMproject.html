manipulation practice project

The DOM manipulation practice project is a way for me (Jacob Curtiss) to practice manipulating the DOM. I want to make this document completely changeable by attaching javascript functions to buttons which interact with the DOM, changing the style and adding content without manually touching the HTML or CSS.
